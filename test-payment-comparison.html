⚡ Technical Implementation

🔄 Unified Flow Pattern:

1. Loading Phase:

  • ✅ Show loading spinner + message
  • ✅ Hide main content containers
  • ✅ API call (charge_qris.php / charge_bank.php)

2. Content Display:

  • ✅ Hide loading after 2 seconds
  • ✅ Show QR code / Virtual Account
  • ✅ Display payment instructions

3. Status Monitoring:

  • ✅ Poll check_status.php every 3 seconds
  • ✅ Update status text in real-time
  • ✅ Handle settlement response

4. Success Handling:

  • ✅ Call complete_payment.php
  • ✅ Change status text to green
  • ✅ Show "Lanjutkan" button
  • ✅ Navigate to selectlayout.php
